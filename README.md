# Quant Trading Platform

⚠️ Educational project only. No real-money trading.

## Overview
This project is a Python-based algorithmic trading and market simulation platform.
It is designed to simulate financial markets, test trading strategies, manage risk,
and analyze performance using historical data.

## Version
v0.1 – Basic market data loader and strategy framework

## Features (Planned)
- Historical market data loading
- Strategy abstraction layer
- Backtesting engine
- Risk management system
- Performance analytics
- Visualization dashboard

## Disclaimer
This project is for educational and research purposes only.
It does NOT provide financial advice or support live trading.
